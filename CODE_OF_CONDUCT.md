# Code of Conduct

## Our Pledge

We aim to keep this project respectful, focused, and welcoming.

## Expected Behavior

- Be direct and constructive.
- Assume good intent.
- Keep discussions technical and relevant.
- Respect privacy and avoid sharing sensitive data.

## Unacceptable Behavior

- Harassment or discrimination.
- Personal attacks.
- Publishing private information without permission.
- Repeated off-topic disruption.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other
contributions that violate this code of conduct.
